Admin Control Panel Manual

1. monitoreo global

Al ingresar como administrador, tendrás acceso al Dashboard Admin, donde podrás ver métricas agregadas de todos tus clientes activos.

  • Comparativa de Clientes: Selecciona múltiples clientes para comparar su rendimiento en un solo gráfico.
  • Métricas Totales: Visualiza el total de mensajes procesados por la infraestructura completa de Optimizar.

2. gestión de clientes

En la sección "Usuarios" puedes crear nuevas cuentas de cliente o editarlas. Cada cliente es independiente.

Configuración Base

  • CSV URL: Fuente principal de datos del cliente.
  • Webhook URL: Endpoint para el chat del cliente.
  • CRM / Agente Externo: Enlaces directos a sus herramientas.

Botones Especiales

Agrega hasta 4 accesos directos personalizados que el cliente verá en su menú superior.

Novedad

3. botones de acción (switches n8n)

Permiten al cliente controlar flujos de n8n mediante interruptores visuales activo/desactivado.

Requerimiento: El workflow de n8n debe manejar:
- GET: Devolver estado actual { "emailEnabled": true }
- POST: Hacer toggle del estado { "action": "toggle" }

Para configurar: añade un switch en el panel admin, ponle un nombre descriptivo y pega la URL del Webhook de n8n.

Avanzado

4. gráficos personalizados

Integra cualquier dato de Google Sheets directamente en el dashboard del cliente con flexibilidad total.

Campos Clave

Columna de Fecha: Nombre exacto de la col. en la Sheet.

Columna de Valor: Columna a medir o dejar "COUNT" para contar registros.

Lógica

Agregación: Suma, Conteo o Promedio.

Tipo: Elige entre Líneas o Barras según el tipo de dato.

IMPORTANTE: La Sheet debe estar publicada en la web como CSV.
URL: .../pub?output=csv
